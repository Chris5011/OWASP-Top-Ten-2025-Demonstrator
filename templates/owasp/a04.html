5. Comparison Cheat Sheet

Method Good For Password Storage? Main Problem / Note
Plaintext Nothing security-related ❌ Never Immediate exposure after database leak
SHA-256 Integrity checks ❌ No Too fast; no per-password uniqueness without salt
Salted SHA-256 Explaining salts ⚠️ Still no Salt helps, but the hash is still too fast
HMAC-SHA-256 Authenticity and tamper detection ⚠️ Not by itself Keyed integrity primitive, not a password hashing function
Argon2id Password storage ✅ Yes Slow and memory-hard by design
