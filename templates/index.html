What is Injection?

Injection happens when untrusted input is interpreted as part of a command, query, or script instead of being handled purely as data.

👤 User Input
→
🌐 Application
→
🗄️ Interpreter

Unsafe Pattern

The application builds commands by concatenating strings.

query = "SELECT ... WHERE user = '" + input + "'"

Problem Input can become logic.

Safe Pattern

The application keeps commands and data separate.

query = "SELECT ... WHERE user = ?"
bind(input)

Fix Input remains data.
